# PowerAutomateDataPipeline
This project goes beyond dashboard development by automating the complete data pipeline. Using Power Automate, Python, VBA, Excel, and Power BI, it extracts, transforms, and updates production data hourly without APIs or database access, delivering reliable insights while eliminating manual reporting and improving operational decision-making.
