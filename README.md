# 🚀 Automated Hourly DHU Analytics Dashboard

> **From Manual Reporting to Fully Automated Hourly Quality Analytics**

This project goes beyond building a Power BI dashboard. The primary objective was to automate the complete data pipeline for hourly production quality reporting, ensuring that management always has access to the latest insights without any manual intervention.

---

## 📌 The Challenge

The management required an **hourly DHU (Defects per Hundred Units)** dashboard to monitor production quality across multiple lines. The dashboard needed to refresh every hour during production for factory managers and leadership to make timely, data-driven decisions.

### The biggest hurdles

- ❌ No database access
- ❌ No APIs
- ❌ Data was only available through a third-party web portal

Manually downloading, cleaning, and updating data every hour was inefficient and unsustainable.

---

## 💡 The Solution

Instead of waiting for API access, I designed an end-to-end automation framework that:

- ✅ Automatically logs into the production portal
- ✅ Navigates to the hourly report
- ✅ Downloads the latest production data
- ✅ Performs ETL and data validation
- ✅ Updates the master dataset
- ✅ Refreshes the Power BI dashboard
- ✅ Sends completion notifications

The entire workflow runs **without human intervention** and is scheduled to execute every hour.

---

## 📊 Dashboard Features

- 📈 Hourly DHU by production line
- 📊 Hourly defect trend analysis
- 📉 Comparison with yesterday's performance
- 📅 Comparison against the previous 7-day average
- 🏆 Best-performing production line
- ⚠️ Lowest-performing production line
- 🔍 Top defect categories driving quality losses

The solution significantly reduced manual reporting effort while providing management with reliable and timely operational insights.

---

## 🎯 Key Takeaway

> **A dashboard is only as valuable as the reliability of the data pipeline behind it.**

Building visuals is only one part of analytics. Designing an automated and dependable reporting process that consistently delivers accurate data is where the real value lies.

---

## 🚀 Future Enhancement

The current solution runs using **Power Automate Desktop**.

The next phase is migrating the workflow to **Power Automate Cloud**, making the automation more scalable, maintainable, and easier to monitor.
